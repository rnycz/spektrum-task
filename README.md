# Spektrum Task

Recruitment task

## Installation and Setup Instructions

Clone down this repository:

```bash
git clone https://github.com/rnycz/spektrum-task.git
```

Make sure that you are in your root folder of application.<br />
Installation:

```bash
npm install
```

Start Server:

```bash
npm start
```

Visit App:

```bash
http://localhost:3000/
```
